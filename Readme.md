# An algorithmic tasks by Aleksei Ivasiuta and Denis Kolpakov

## Usage

### TypeScript

- Open a directory
- Run `index.ts` file using `ts-node`

```
cd ./remove_duplicates_from_sorted_array
./ts-node ./index.ts
```

### Go

- Open a directory
- Run `main.go` file using `go run`

```
cd ./remove_duplicates_from_sorted_array
go run ./main.go
```
